# retro

`retro` is a core library for Dofus Retro. It declares data types and provides constants and functions to work with the
game. It also declares the Storer interface implemented in package [retropg](https://github.com/hadamrd/retrodatapg).

## Requirements

- [Git](https://git-scm.com/)
- [Go](https://golang.org/)

## Installation

```sh
go get -u -v github.com/hadamrd/retrodata
```
